3 formas de ligar o CSS no HTML

Forma Como funciona Exemplo Vantagem / Desvantagem
Inline Dentro da tag HTML p style="focolor:red;">Texto Rápido, mas desorganizado
Interno Dentro da tag "style" no HTML style p { color:red; } style Bom para páginas pequenas
Externo Em um arquivo separado (style.css) Link rel="stylesheet" href="style.css" Mais usado — organizado e reutilizável
