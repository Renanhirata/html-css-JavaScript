A “Mira” (Seletores Principais)

Tipo Símbolo Exemplo O que faz
TAG nenhum p { color: red; } Estiliza todas as tags

ID # #menu { background: blue; } Estiliza um elemento específico
CLASSE . .noticia { font-size: 18px; } Estiliza vários elementos com a mesma classe
Universal * * { margin: 0; } Estiliza todos os elementos da página
